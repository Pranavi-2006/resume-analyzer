# 🤖 AI Resume Analyzer

## 📌 Description
This is a Python-based Resume Analyzer that evaluates resumes using keyword matching and provides suggestions to improve resume quality.

## 🚀 Features
- Skill detection
- Resume scoring
- Section validation (Education, Skills, Projects)
- Feedback and suggestions

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Add your resume in `resume.txt`
2. Run the program:
   python main.py

- Add NLP (Natural Language Processing)
